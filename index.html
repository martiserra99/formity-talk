Formity

Create dynamic React forms with just a JSON

Created by Mart√≠ Serra Molina

Problem

There is no easy way to create dynamic forms in React. Dynamic forms are forms of multiple steps, in which the questions in each step depend on the answers given in the previous ones.

Solution

Formity solves this problem by allowing you to create dynamic forms with just a JSON. In this JSON you can define all kinds of logic using variables, conditions, loops and operators.

                    "defaultValues": {
                      "name": "",
                      "surname": ""

                    ]
                  }

                  "loop": {

                      { "variables": { "sport": { "$arrayElemAt": ["$sports", "$i"] } } },
                      {
                        "variables": {
                          "question": {
                            "$getField": { "field": "$sport", "input": "$questions" }
                          }
                        }
                      },
                      {
                        "form": {
                          "defaultValues": { "days": [] },

                            "days": [
                              [{ "_$ne": [{ "_$size": "_$days" }, 0] }, "At least one day"]
                            ]
                          },

                            {
                              "LayoutForm": {
                                "heading": "$question",
                                "text": "Select the days",
                                "fields": [
                                  {
                                    "CheckboxGroup": {
                                      "name": "days",
                                      "label": "Days",
                                      "list": [
                                        { "value": "monday", "label": "Monday" },
                                        { "value": "tuesday", "label": "Tuesday" },
                                        { "value": "wednesday", "label": "Wednesday" },
                                        { "value": "thursday", "label": "Thursday" },
                                        { "value": "friday", "label": "Friday" },
                                        { "value": "saturday", "label": "Saturday" },
                                        { "value": "sunday", "label": "Sunday" }
                                      ]
                                    }
                                  }
                                ],
                                "buttons": [
                                  { "Back": { "children": "Back" } },
                                  {
                                    "Button": {
                                      "type": "submit",
                                      "children": "Next"
                                    }
                                  }
                                ]
                              }
                            }
                          ]
                        }
                      },
                      {
                        "variables": {
                          "sports": {
                            "$map": {
                              "input": "$sports",
                              "as": "sport",
                              "in": {
                                "$cond": {
                                  "if": { "$eq": ["$$sport", "$sport"] },
                                  "then": { "sport": "$$sport", "days": "$days" },
                                  "else": "$$sport"
                                }
                              }
                            }
                          },
                          "i": { "$add": ["$i", 1] }
                        }
                      }
                    ]
                  }
                },
                {
                  "return": {
                    "name": { "$concat": ["$name", " ", "$surname"] },
                    "age": "$age",
                    "drivingLicense": "$drivingLicense",
                    "sports": "$sports"
                  }
                }
              ]

Advantages

Since Formity allows you to define these forms as JSON objects, you can store them as JSON files, send them through the network and even store them in a database like MongoDB.

Mongu

Formity depends on the package Mongu. It is a package that lets you use the expressions you would use in MongoDB, but in JavaScript. 

            
              const { mongu } = require('mongu');

              const expr = {
                fullName: {
                  $concat: [{ $toLower: '$name' }, ' ', { $toLower: '$surname' }],
                },
                isAdult: { $gte: ['$age', 18] },
              };
              
              const vars = { name: 'Marti', surname: 'Serra', age: 24 };
              
              console.log(mongu(expr, vars)); // { fullName: "marti serra", isAdult: true }
            
          
            
              const { mongu } = require("mongu");

              const expr = { $add: ["$a", "$b"] };
              
              const vars = { a: 5, b: 10 };
              
              console.log(mongu(expr, vars)); // 15
            
          
            
              const { mongu } = require("mongu");

              const expr = {
                $cond: {
                  if: { $gte: ["$age", 18] },
                  then: "You are an adult",
                  else: "You are not an adult",
                },
              };
              
              const vars = { age: 18 };
              
              console.log(mongu(expr, vars)); // "You are an adult"
            
          
            
              const { mongu } = require("mongu");

              const expr = {
                $map: {
                  input: "$numbers",
                  as: "num",
                  in: { $multiply: ["$$num", 5] },
                },
              };
              
              const vars = { numbers: [1, 2, 3] };
              
              console.log(mongu(expr, vars)); // [5, 10, 15]

Components

To use Formity we will create the components that our forms will use, and we will provide them using the FormityProvider component.

Form

To create a form we will use the Formity component. This component receives a JSON that defines the form and a function that will be called when the form is submitted.

JSON

The JSON that defines the form is an array of objects. Each object can be of type form, variables, cond, loop or return.
